# container-migrator

## 测量网速
```iperf
iperf -s
ierf -c ip
```


```server
./container-migrator/migrator server --migrated_container_dir /home/ubuntu/target
./migrator server --migrated_container_dir /opt/container-migrator/server_repo/targetv0.1
```

```
cd workloads/redis
mkdir -p rootfs
docker export $(docker create redis) -o file.tar
tar -C rootfs -xf file.tar
rm -f file.tar
```


```
mount -t tmpfs none <dir>
```
